## What is Impuu?
- This is a project that intends to automate the process of make TAX.

---
## What problem does Impuu solve ?
- There are different software that help people to do taxes, but they have a high price or they are very difficult tools to use. The objective of Impuu is to be very easy to use and to be able to help users to prepare their taxes from any Country..

### Developer Setup:
- Fork this repository, and cd into it.
```bash
git clone https://github.com/impuu/backend.git
cd Impuu/
```
- Create and activate your virtual environment.
    - MacOS/Linux:
    ```bash
    virtualenv --no-site-packages env
    source env/bin/activate
    ```
    - Windows:
    ```
    virtualenv env
    .\env\Scripts\activate
    ```
- Install requisite python packages and modules.
```bash
pip install -r requirements.txt
```
- Verify if the Django project is working by running the development server.
```bash
python manage.py runserver
```
